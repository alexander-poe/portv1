InstaFrench

  • Overview: A flashcard like approach to learn French using a spaced repetiton algorithm.

    Source Live

  • Challenge: The challenge of this project was to create a simple UI that would represent a flashcard like approach to learning. While this technology can be reused in many ways, some of the biggest difficulties faced in this project was backend authentication with Oauth and bearer. My partner took on this part of the project initially and we got to a place where we were heavily stuck. I managed to come in with a fresh set of eyes and essentially solve the problem by installing react-cookies and sending the authentication when dispatching an action to the server. Example on next slide.

  • Tech: Essentially with passports google Oauth a user signs in with their google information and a authentication token is stored in the cookies. Bearer, used for route authentication requires the authenticatin token in cookies for access, so in this dispatch to server, without the cookie being sent in headers, bearer would do its job and not let that happen.

  • Takeaways: This project was a fun with a lower level of complexity and moving parts compared to the projects I am working on now. I'd like to explore making this entire project dynamic in a way that a user could easily customize the theme of the project and the questions it asks. That way it could be a very useful studying tool for any subject as opposed to the one we chose.
